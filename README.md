## About

Momentum is a modern web app for generating and managing customizable countdown
cards. Built as a learning project to explore and master current web
technologies, it showcases experience with:

- **TypeScript** (previously JavaScript): safer, scalable code!
- **React**: fast UI development and reusable components.
- **Deno** & **Node.js**: demonstrates back-end versatility.
- **npm**: modern dependency management and build workflows.
- **PWA**: installable, offline-ready, mobile-friendly experience.
- **GitHub Actions (CI/CD)**: automated building, testing, and deployment.
- **HTML & CSS**: accessible, responsive, and visually pleasing interface.

---

## Tech Stack

- **Frontend:** React, TypeScript, Vite, CSS Modules.
- **Build Tools:** npm, Vite.
- **Back-end/Tasks:** Deno (for task automation), Node.js.
- **PWA:** Service Worker, Manifest JSON, Home Screen installability.
- **CI/CD:** GitHub Actions for automated deployment to GitHub Pages.
